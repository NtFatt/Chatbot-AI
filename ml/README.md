# Low Level 4 ML Workspace

This workspace contains scripts and configurations for the Low Level 4 (L4) local LoRA/SFT training pipeline.

## Capabilities
- Train LoRA adapters using `peft` and `trl`
- Serve fine-tuned adapters using FastAPI

Please refer to `docs/LOW_LEVEL_4_LOCAL_LORA.md` for full instructions.
